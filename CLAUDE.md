# AetherHub

Job & Talent Management Platform for KOL Specialists.

## Tech Stack

- **Frontend**: Next.js 14, ShadCN UI, TailwindCSS
- **Backend**: Go (Fiber/Gin)
- **Database**: Supabase (PostgreSQL)

## Commands

### Frontend
```bash
cd frontend
npm run dev        # Start dev server
npm run build      # Build for production
npm run lint       # Run ESLint
npm test           # Run tests
```

### Backend
```bash
cd backend
go run cmd/server/main.go   # Start dev server
go build ./...               # Build
go test ./...                # Run tests
go vet ./...                 # Lint
```

### Docker
```bash
docker compose up -d         # Start all services
docker compose down          # Stop all services
docker compose logs -f       # View logs
docker compose build         # Rebuild images
```

## Architecture

```
aetherhub/
├── frontend/          # Next.js app with ShadCN UI
│   ├── app/           # App router pages
│   ├── components/    # React components
│   └── lib/           # Utilities
├── backend/           # Go API server
│   ├── cmd/           # Entry points
│   ├── internal/      # Business logic
│   │   ├── handlers/  # HTTP handlers
│   │   ├── models/    # Data models
│   │   ├── services/  # Business services
│   │   └── middleware/# HTTP middleware
│   └── migrations/    # Database migrations
├── .claude/           # Claude Code config
│   ├── prds/          # Product Requirements Documents
│   └── plans/         # Implementation plans
└── supabase/          # Supabase config
```

## Roles

- **Superadmin**: Platform owner, manages all users
- **Admin (KOL Specialist)**: Manages jobs, assigns talents, monitors progress
- **Talent**: Views assigned jobs, updates progress
- **Brand**: Accesses via unique code, approves drafts

## Key Workflows

### Job Progress Flow
1. Absen (attendance)
2. Draft Storyline → Draft Content (optional)
3. Input Link Postingan
4. Insight (manual input by talent)

### Brand Access
- Brand accesses via unique code (no traditional auth)
- Code generated by Admin for each job
